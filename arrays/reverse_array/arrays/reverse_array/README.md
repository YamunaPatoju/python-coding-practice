# Reverse Array
## Problem Statement
Given an array, reverse all its elements.
## Example
Input:
[1,2,3,4,5]
Output:
[5,4,3,2,1]

## Approach
Use two pointers:
1. left starts from the beginning of the array.
2. right starts from the end of the array.
3. Swap the elements at left and right.
4. Move left forward.
5. Move right backward.
6. Continue until both pointers meet.
