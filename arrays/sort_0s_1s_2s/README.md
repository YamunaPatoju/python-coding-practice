Sort an Array of 0s, 1s and 2s

Problem Statement-

Given an array consisting only of 0s, 1s and 2s, sort the array in ascending order.

Example-

Input:

[0, 2, 1, 2, 0, 1]

Output:

[0, 0, 1, 1, 2, 2]

Approach-

1.Create three separate lists:

-zero for storing 0s

-one for storing 1s

-two for storing 2s

2.Traverse the array.

3.Add each element to its corresponding list.

4.Combine the three lists.

5.Update the original array.
