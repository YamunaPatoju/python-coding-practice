# Cyclic Rotation of an Array by One

## Problem Statement

Given an array, cyclically rotate the array clockwise by one position.

## Example

**Input:**

[1, 2, 3, 4, 5]

**Output:**

[5, 1, 2, 3, 4]

## Approach

1. Store the last element of the array.
2. Shift all elements one position to the right.
3. Place the stored last element at the first position.
4. Return the rotated array.
