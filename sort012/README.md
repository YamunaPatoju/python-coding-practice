Sort an Array of 0s, 1s and 2s

Problem Statement-

Given an array consisting only of 0s, 1s and 2s, sort the array in ascending order.

Example-

Input:

[0, 2, 1, 2, 0, 1]

Output:

[0, 0, 1, 1, 2, 2]

Approach-

1.Create three separate lists:

2.zero for storing 0s

3.one for storing 1s

4.two for storing 2s

5.Traverse the array.

6.Add each element to its corresponding list.

7.Combine the three lists.

8.Update the original array.
