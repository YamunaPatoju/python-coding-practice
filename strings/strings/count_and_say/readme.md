# Count and Say

## Problem Statement

The count-and-say sequence is generated by describing the previous term.

- countAndSay(1) = "1"
- countAndSay(n) is obtained by reading countAndSay(n-1).

Return the nth term of the sequence.

## Example 1

Input:

```text
n = 4
```

Output:

```text
"1211"
```

Explanation:

```text
1
11
21
1211
```

## Example 2

Input:

```text
n = 1
```

Output:

```text
"1"
```

## Approach

- Start with "1".
- Generate the next term by counting consecutive digits.
- Append count followed by the digit.
- Repeat until the nth term is generated.
